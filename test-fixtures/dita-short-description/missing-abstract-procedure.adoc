= Enabling AWQ quantization

Follow these steps to enable AWQ quantization.

.Prerequisites

* A model with AWQ quantization available
* vLLM 0.4.0 or later

.Procedure

. Download the AWQ-quantized model:
+
[source,bash]
----
huggingface-cli download TheBloke/Llama-2-7B-AWQ
----

. Run the server with quantization:
+
[source,bash]
----
vllm serve --model TheBloke/Llama-2-7B-AWQ --quantization awq
----

.Verification

* Check memory usage to confirm quantization is active.
