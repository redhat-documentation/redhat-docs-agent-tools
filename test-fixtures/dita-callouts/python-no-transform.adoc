== Using the vLLM Python API

[source,python]
----
from vllm import LLM, SamplingParams  # <1>

llm = LLM(model="meta-llama/Llama-2-7b")  # <2>
sampling_params = SamplingParams(temperature=0.8)  # <3>

outputs = llm.generate(["Hello, world!"], sampling_params)  # <4>
print(outputs[0].outputs[0].text)
----
<1> Import the LLM and SamplingParams classes
<2> Initialize the LLM with the model name
<3> Configure sampling parameters
<4> Generate text completions
