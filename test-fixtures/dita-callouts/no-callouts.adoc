= Basic installation

[role="_abstract"]
Install vLLM using pip.

.Procedure

. Create a virtual environment:
+
[source,bash]
----
python -m venv vllm-env
source vllm-env/bin/activate
----

. Install vLLM:
+
[source,bash]
----
pip install vllm
----

. Verify the installation:
+
[source,bash]
----
python -c "import vllm; print(vllm.__version__)"
----
