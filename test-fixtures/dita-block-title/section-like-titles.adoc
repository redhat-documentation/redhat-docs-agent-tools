.VM snapshot controller and custom resources

The VM snapshot feature introduces three new API objects defined as custom resource definitions (CRDs) for managing snapshots. These objects provide a way to create, manage, and restore snapshots of virtual machines.

.Configuration methods

You can configure the feature using one of the following methods. Each method has different advantages depending on your deployment scenario and requirements.
