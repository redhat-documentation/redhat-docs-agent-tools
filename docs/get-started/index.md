# Get Started with Cursor

Use these pages to get started working with **Red Hat Docs Agent Tools** in **Cursor**.

Start by reading [Cursor fundamentals](cursor-fundamentals.md) to learn more about Cursor's Agent panel, loading `AGENTS.md`, terminology, and naming rules.

Then, based on your goals, pick a guide below.

- **[Use Cursor and these tools with your product documentation](cursor-product-documentation.md)** — Learn how to use Cursor with these agentic tools if your AsciiDoc or Markdown source code is located in another Git repository.
- **[Contribute to this repo by using Cursor](../contribute/cursor-contributing-tools.md)** — Learn how to use Cursor to contribute code to this repository.
